# Predictive HR Analytics with External Validation

Code for the paper: **What Drives Employee Productivity and Organizational Competitiveness? Evidence from Predictive HR Analytics with External Validation**

## Description
This repository contains all the code used in the paper, including data preprocessing, model training (OLS, CART, RF, XGBoost), evaluation metrics, and SHAP interpretability. It uses the public IBM HR dataset from Kaggle and synthetic data for the confidential company dataset.
